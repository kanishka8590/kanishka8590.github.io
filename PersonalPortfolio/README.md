# Personal Portfolio Website

## Overview
A responsive personal portfolio website built with HTML, CSS, and JavaScript designed to showcase my web development expertise, academic projects, and professional journey effectively. The website showcases her **skills, projects, experience, and contact information** in a modern, interactive, and visually appealing layout.  

The site features a interactive **skills bubbles**, project cards, and smooth scrolling navigation for an engaging user experience.


---

## Features
- **Hero Section:** Typewriter effect for "Web Developer | Problem Solver | Programmer".  
- **About Me:** Refined multi-line description with circular profile photo and shadow.  
- **Skills:** Interactive **bubble-style** display of programming and web development skills.  
- **Projects:** Cards for each project, including web apps, games, and AI projects.  
- **Experience:** Timeline of internships with CodeClause and Prodigy Infotech.  
- **Contact Section:** Email, phone, GitHub, and LinkedIn links.  
- **Smooth Scroll Navigation:** Navigation links scroll smoothly to sections.  
- Fully **responsive design** for mobile and desktop.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Animations:**  typewriter effect, hover interactions  
- **Responsive Design:** Flexbox and media queries

# 🎮 Tic Tac Toe Web App  

An interactive **Tic Tac Toe** game built with **HTML, CSS, and JavaScript**.  
Choose **X or O**, play in **Single Player (vs AI)** or **Two Player mode**, and enjoy fun win/lose effects with a modern, responsive design.  

---

## ✨ Key Features  
- 🎯 **Marker Selection** → Choose to play as X or O  
- 👥 **Modes** → Single Player (AI) or Two Player    
- 🎉 **You Win!** → Confetti popup celebration  
- 😢 **You Lose** → Sad falling tears animation  
- 📊 **Score Tracking** → Keeps track of wins  

---

## 🛠️ Technologies  
- **HTML5** – Game structure  
- **CSS3** – Styling & animations  
- **JavaScript (ES6)** – Game logic & AI  
- **Canvas Confetti** – Win animations  

---

## Usage

1. Clone the repostory
2. Open `index.html` in a web browser.  
3. Select your marker and player mode, First to align 3 markers wins!
